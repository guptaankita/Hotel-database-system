Abstract A konak is a hotel management database which enables user to view/book hotel room in any country. The user can search hotels by various criteria like check-in
date, check-out date,number of rooms,rating, numbers of childerns and adults.We have user registration facility after which they can book the hotel room from the site. Administrator has various functions like delete registered user,cancel hotel booking and delete review.
Introduction The purpose was to simplify the hotel booking by allowing the user sitting at the convenience of his home to search hotels that match his requirement from a huge database that contains information about a wide range of hotels and then book a hotel of his choice. Objective of the application are as follows:
1.Application through which customers have access to a hotel database and can book and review hotels.
2.Application will be intuitive to use and easy-to-maintain.
3.Application will be able to handle simultaneous requests from multiple clients.
4.Application will be designed using client server architecture.

Implementation We have queried a web service by invoking it through its given API. Passing the required parameters & the hotel search results data was transmitted back in either XML or JSON format. We have set the output type to JSON. We have passed the JSON data extracting only those attributes which are useful for the project. We have implemented the web application in Eclipse IDE using JSF framework, it is a Java specification for building component-based user interfaces for web applications. Thus our front end web pages being developed in JSP pages, our business tier logic was implemented using Java. We have thus cleanly separated our Client layer, Presentation & Business Tier. We have used an open source MySQL(5.1) for storing our data. We have used Navicat tool UI tool for interacting with the database.
Discussion Konak makes sure that User can perform the following operations like booking a hotel,searching a hotel by the specified criteria, add a review.
Administrator can Delete reviews
Cancel booking made by a user
Delete a user.
